03
Project Idea

Reasoning is not the whole story

What most LLM work emphasizes

Reasoning ability, task automation, tool use, coding, math, and instruction following.

What affective language requires

Detecting, tracking, and expressing emotional state under context and personality.

Large commercial models often imitate emotional nuance because they have enormous training scale. Smaller local models have fewer stored patterns, so emotional behavior needs stronger external structure.

Synthetic Data

Why generate EmoDynamic?

1. Existing datasets rarely provide dialogue turns with continuous VAD labels for both speakers over time.
2. We generate scenario + persona + turn-level VAD data using a strong teacher model.
3. The logic is similar to distillation: transfer high-capacity teacher behavior into smaller supervised components.

Section 2

GRU: recurrent emotion dynamics

Dialogue emotion is not a single utterance problem. Each new turn updates the speaker’s internal emotional state.

A GRU is useful because it repeatedly updates a hidden state while deciding what to keep, reset, or overwrite.

In Co-EmP, the hidden state is treated as the speaker’s evolving emotional embedding.

GRU training setup

Dataset

Section 3

LLM generator: prompt as the control interface

Affect block

  • Valence controls pleasantness: hurt ↔ warm.
  • Arousal controls energy: quiet ↔ urgent.
  • Dominance controls social force: hesitant ↔ firm.
  • Few-shot examples translate numeric VAD into spoken tone.
Important: the model never says the numbers aloud. The VAD only guides style.

Generation rules

  • Write only the next spoken reply.
  • Stay consistent with scenario, private context, and personality.
  • No analysis, labels, JSON, stage directions, or multiple candidates.
  • Keep turns realistically short.

Conclusion

Conclusion

1. Detect

Use a VAD regressor to ground text in continuous affective space.

2. Track

Use GRU dynamics to update emotional state across dialogue turns.

3. Generate

Condition a small LLM with personality and target affect.

Core claim: small LLMs can become more emotionally consistent when affect is modeled explicitly rather than left entirely implicit.
